Projects

YouTube to MP3 Converter

A lightweight Flask-based web application that allows users to convert YouTube videos into high-quality MP3 files using yt-dlp and FFmpeg. The app securely handles audio extraction on the backend and presents a clean, responsive user interface. It ensures smooth performance, quick processing, and an intuitive experience for users looking for fast and reliable audio conversions.

FaceSense – Real-Time Emotion Detector

A Python-based app using OpenCV and DeepFace to detect human emotions via webcam in real time. It tracks faces, processes expressions like joy or anger, and displays live predictions using AI. Built with a focus on performance, it showcases real-time computer vision and AI integration skills.

DictSuper – Offline Dictionary Web App

A React-based offline-first dictionary that fetches word definitions via API and caches them for future use. It features a smart Google-style search bar with live autocomplete using a 100K+ word list. Styled with Tailwind CSS and powered by IndexedDB, it demonstrates my skills in building fast, responsive, and installable web apps with real-world functionality.
